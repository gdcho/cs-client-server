This sample app shows a pure javascript based Single Page App that uses a lightweight trivia web api.
Copy triviaapi under webapps.
Restrat tomcat.
From the browser invoke http://localhost:8081/triviaapi/index.html
Press Login button (you dont have to type in username and password).
You will now see a file upload shows up.  Upload a file.
Press List button.  You will see file listing shows up just above it.
Press Delete button, you will see some pop up instructions.
Check the tomcat console log.  Stdy indexhtml and the two java files in the classes folder.
